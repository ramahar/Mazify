Maze Generation

Maze generation algorithms are automated methods for the creation of mazes

These mazes are equivalent to a tree in graph theory. Various maze solving algorithms will be demonstrated while searching for a path from a starting node to an end node.


A* search algorithm

This maze is generated using Prim's Algorithm. A* is used to solve the maze

Maze Solving

Now we can explore some maze solving algorithms

Depth-first search is used in this example. DFS is an algorithm for traversing graph data structures. The algorithm starts at the root node and explores as far as possible along each branch before backtracking.
